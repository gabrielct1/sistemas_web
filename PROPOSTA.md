# Timer Pomodoro

## Discente: Gabriel da Cunha Torres

### Resumo
Esse trabalho tem como objetivo o desenvolvimento de um sistema web de aplicativo para o método de estudos Pomodoro. No sistema, o usuário poderá definir os tempos para estudos e descanso. O sistema terá uma interface limpa, com o intuito de eliminar distrações.

### Escopo
Este projeto terá as seguintes funcionalidades:
- Login de usuários
- Definição de tempos personalizados, além do número de pomodoros até um descanso longo.
- Possibilidade de escutar música sem letras enquanto estuda.
- Ver um relatório do tempo gasto estudando.

### Restrições
Neste trabalho não serão consideradas:
- Usuário escolher as músicas que estarão tocando no fundo, apenas se quer ou não que toquem.
- Funcionalidades de login avançadas.
- Interação entre usuários.
